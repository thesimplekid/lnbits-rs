# lnbits-rs


An ergonomic, [LNbits](https://lnbits.com/) API Client for Rust.



## Key features

- [x] Create invoices
- [x] Webhook for incoming invoices
- [x] Decode invoices
- [x] Pay invoices
- [x] Get wallet details
- [x] Tor support
